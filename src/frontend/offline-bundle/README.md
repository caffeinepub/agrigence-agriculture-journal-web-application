# Agrigence Offline Website Bundle

This is a complete, standalone offline website that works by simply opening `index.html` in any modern web browser. No server or internet connection required.

## How to Use

1. **Download** this entire folder to your computer
2. **Navigate** to the folder location
3. **Double-click** `index.html` to open the website in your default browser
4. Alternatively, right-click `index.html` and choose "Open with" to select a specific browser
